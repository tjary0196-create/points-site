# اختبار شامل للنظام 🧪

## قائمة الاختبار

### 1. نظام الملف الشخصي ✅
- [ ] تسجيل دخول المستخدم
- [ ] الانتقال إلى "ملفي الشخصي"
- [ ] إضافة يوزرنيم فريد
- [ ] التحقق من توفر اليوزرنيم
- [ ] رفع صورة شخصية
- [ ] إضافة بيانات الاتصال
- [ ] حفظ التغييرات
- [ ] التحقق من ظهور البيانات في البطولات

### 2. نظام المزاد 🔨
- [ ] الانتقال إلى "متجر الألعاب والاشتراكات"
- [ ] اضغط "+ إنشاء مزاد"
- [ ] إدخال تفاصيل الحساب
- [ ] إدخال السعر الابتدائي والأدنى
- [ ] تحديد مدة المزاد
- [ ] رفع 5 صور
- [ ] إنشاء المزاد
- [ ] التحقق من ظهور المزاد في القائمة
- [ ] اختبار تمرير الصور (يمين/يسار)
- [ ] عرض تفاصيل المزاد
- [ ] وضع عرض سعر جديد
- [ ] التحقق من تحديث السعر الحالي

### 3. نظام المستويات 🏆
- [ ] التحقق من عرض مستوى المستخدم الحالي
- [ ] عرض شريط التقدم
- [ ] التحقق من حساب المستوى بناءً على الإنفاق
- [ ] الانتقال من برونزي إلى فضي (50$)
- [ ] الانتقال من فضي إلى ذهبي (150$)
- [ ] عرض الرصيد المتبقي للمستوى التالي

### 4. أكواد الشحن 🔑
- [ ] الدخول إلى لوحة الأدمن
- [ ] الذهاب إلى "أكواد الشحن"
- [ ] توليد 10 أكواد بقيمة 10$ لكل واحد
- [ ] التحقق من ظهور الأكواد
- [ ] نسخ أحد الأكواد
- [ ] الذهاب إلى "محفظتي"
- [ ] إدخال الكود
- [ ] التحقق من إضافة الرصيد
- [ ] محاولة استخدام نفس الكود مرة أخرى (يجب أن يرفع)
- [ ] التحقق من تحديث حالة الكود في لوحة الأدمن

### 5. الإعلانات المدفوعة 📢
- [ ] الدخول إلى لوحة الأدمن
- [ ] الذهاب إلى "الإعلانات المدفوعة"
- [ ] رفع صورة إعلان
- [ ] إضافة رابط
- [ ] إضافة نص الشريط المتحرك
- [ ] حفظ الإعلان
- [ ] الذهاب إلى المتجر
- [ ] التحقق من ظهور الإعلان
- [ ] التحقق من ظهور شريط الإعلانات المتحرك
- [ ] اختبار الرابط

### 6. نظام الإحالة ✅
- [ ] تسجيل دخول المستخدم الأول
- [ ] نسخ رابط الإحالة
- [ ] تسجيل دخول جديد بمستخدم آخر
- [ ] استخدام رابط الإحالة
- [ ] التحقق من ربط المستخدم الجديد بالمحيل
- [ ] إجراء عملية شحن من المستخدم الجديد
- [ ] التحقق من إضافة المكافأة للمحيل
- [ ] سحب المكافأة المعلقة
- [ ] التحقق من تحويل المكافأة إلى الرصيد

### 7. لوحة الأدمن 👨‍💼
- [ ] الدخول بحساب أدمن
- [ ] عرض نظرة عامة
- [ ] البحث عن مستخدم
- [ ] عرض بيانات المستخدم
- [ ] تعديل رصيد المستخدم
- [ ] عرض المزادات النشطة
- [ ] البحث عن مزاد
- [ ] إغلاق مزاد
- [ ] عرض أكواد الشحن
- [ ] عرض الأكواد المستخدمة
- [ ] إدارة إعدادات المستويات
- [ ] تحديث حدود المستويات

### 8. التأثيرات البصرية ✨
- [ ] التحقق من توهج النيون على العناوين
- [ ] التحقق من حركة الشعارات
- [ ] التحقق من انزلاق العناصر
- [ ] التحقق من تأثيرات التحويم
- [ ] التحقق من تأثيرات الألوان المتدرجة
- [ ] التحقق من سلاسة الحركات

### 9. الأداء والسرعة ⚡
- [ ] قياس سرعة تحميل الصفحة
- [ ] التحقق من عدم تأخر الواجهة
- [ ] اختبار تحميل الصور الكبيرة
- [ ] التحقق من استجابة الأزرار
- [ ] اختبار الحركات السلسة

### 10. التوافقية والمتصفحات 🌐
- [ ] اختبار على Chrome
- [ ] اختبار على Firefox
- [ ] اختبار على Safari
- [ ] اختبار على الهاتف المحمول
- [ ] اختبار على الأجهزة اللوحية

### 11. الأمان 🔒
- [ ] التحقق من عدم إمكانية استخدام كود الإحالة الخاص بك
- [ ] التحقق من عدم إمكانية استخدام نفس الكود مرتين
- [ ] التحقق من عدم إمكانية تعديل رصيد الآخرين
- [ ] التحقق من حماية البيانات الحساسة

### 12. معالجة الأخطاء 🛡️
- [ ] إدخال بيانات خاطئة
- [ ] قطع الاتصال بالإنترنت
- [ ] انتهاء صلاحية الجلسة
- [ ] محاولة الوصول لصفحات محظورة
