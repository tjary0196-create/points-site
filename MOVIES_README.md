# 🎬 قسم الأفلام - دليل شامل

## نظرة عامة
قسم الأفلام المتقدم يوفر تجربة سينمائية احترافية تشبه Netflix مع 7 سيرفرات مشاهدة مجانية، نظام اشتراكات مميز، وجلب تلقائي للأفلام.

---

## 🚀 المميزات الرئيسية

### 1. تصميم Netflix الاحترافي
- واجهة عصرية وجذابة
- شريط أفلام متحرك (Carousel)
- صور عالية الجودة للأفلام
- تأثيرات بصرية متقدمة

### 2. سبعة سيرفرات مشاهدة
```javascript
1. VidSrc.to (الأسرع والأقوى)
2. VidSrc.me
3. 2Embed.cc
4. Embed.su
5. AutoEmbed.to
6. MultiEmbed.mov
7. Player.vdocipher.com (احتياطي)
```

### 3. نظام الاشتراكات المميز (Premium)
- **السعر**: 5,000 ل.س / شهرياً
- **المميزات**:
  - إزالة جميع الإعلانات
  - سيرفرات خاصة وسريعة
  - جودة 4K بدون تقطيع
  - دعم فني 24/7

### 4. جلب تلقائي للأفلام
- تكامل مع TMDB API
- جلب آلاف الأفلام المترجمة
- تحديث يومي للمحتوى
- أوصاف احترافية للأفلام

### 5. نظام الإعلانات المتقدم
- إعلانات Google AdSense
- مساحات إعلانية مدفوعة
- إعلانات متجاوبة
- خيارات متعددة للناشرين

## 🔧 التثبيت والإعداد

### 1. استخراج مفتاح TMDB API
```bash
# توجه إلى https://www.themoviedb.org/settings/api
# سجل حساباً جديداً
# احصل على API Key
# استبدل المفتاح في movies-api.js
```

### 2. تحديث مفتاح AdSense
```javascript
// في movies.html
// ابحث عن: class="ad-banner"
// ضع شفرة AdSense الخاصة بك
```

### 3. تخصيص السعر
```javascript
// في movies.html
// ابحث عن: <div class="premium-price">
// غيّر السعر حسب رغبتك
```

## 📊 كيفية العمل

### جلب الأفلام

```javascript
// movies-api.js
async function fetchMovies(endpoint, containerId) {
    // يجلب الأفلام من TMDB
    // يعرضها في شبكة متجاوبة
    // يربطها مع السيرفرات السبعة
}
```

### المشاهدة

1. المستخدم ينقر على فيلم
2. يفتح نافذة المشغل (Player Modal)
3. يختار السيرفر المفضل
4. يبدأ المشاهدة مباشرة

### الاشتراك المميز

```javascript
// عند الاشتراك:
// 1. يتم خصم المبلغ من رصيد المستخدم
// 2. يتم تفعيل حقل isPremium في Firestore
// 3. تختفي الإعلانات تلقائياً
// 4. يحصل على سيرفرات خاصة
```

## 💰 نظام الأرباح

### من الإعلانات
- **Google AdSense**: 0.5 - 5 دولار لكل 1000 مشاهدة
- **Adsterra**: 1 - 10 دولار لكل 1000 مشاهدة
- **PropellerAds**: 0.5 - 8 دولار لكل 1000 مشاهدة

### من الاشتراكات
- **السعر**: 5,000 ل.س / شهرياً
- **العمولة**: 70% للموقع، 30% للمنصة

### الحد الأدنى للسحب
- **Google AdSense**: 100 دولار
- **Adsterra**: 5 دولار
- **PayPal**: 10 دولار
